drupal.org

Screenshot of drupal.org.

Some important URLs to try and remember. Commonly referred to as d.o. is the central site for all things Drupal.

Documentation

Photo CC-BY Emma Jane Hogbin Westby.

Slash documentation. Guides for pretty much everything, highlight /documentation/git

Projects

Slash project and the short name of the project you are looking for. This doesn’t always match module name, display suite as an example /ds

Support

Slash support. Forums, Community Documentation, Language Specific communities, Books, Professional Service Providers, Training providers

Chat

IRC which is linked from support is important for those contributing to Drupal, I will elaborate on this in a little bit.

API

Allows you to search the complete Drupal Application Programming Interface docs for Drupal 4.6 through 8, this will become indispensable if you are working on core code or you start developing modules.

Drupal Association

Helps the community with funding, infrastructure, education and more. You should become a member tonight if you aren’t already.. for as little as $15USD.

Groups

g.d.o is a place for groups to organise, meet, and work on projects based on interest or geographic location.

Jobs

If you want to advertise a position or are looking for work, this is where you should be looking, but also check g.d.o.

Events

Aggregates all Drupal events that have been posted to g.d.o and displays them on an interactive map.

Register

To make the most of these sites you’ll want a d.o account. If you don’t already have a d.o account let's create one now, and/or update your profile. Also customise your dashboard.

Issue queue

Issue queues are essentially the community todo list, all projects have them.

Dreditor

It’s where you can point out bugs or suggest improvements to Drupal core, and people can test and fix them.

Git

Makes the issue queue easier to work with, go ahead and install it now. Find an issue with a patch, click the review button, click the simply test me button.
