Application Programming Interface

Screenshot of Drupal API.

Allows you to search the complete Drupal Application Programming Interface docs for Drupal 4.6 through 8, this will become indispensable if you are working on core code or you start developing modules.

Jobs

If you want to advertise a position or are looking for work, this is where you should be looking, but also check g.d.o.

Register

drupal.org/user/register

Screenshot of new user account creation.

To make the most of these sites you’ll want a d.o account. If you don’t already have a d.o account let's create one now, and/or update your profile. Also customise your dashboard.

Issue queue

drupal.org/project/issues/drupal

Screenshot of Drupal core issue queue.

Issue queues are essentially the community todo list, all projects have them. It’s where you can point out bugs or suggest improvements to Drupal core, and people can test and fix them.

Filtered search

drupal.org/project/issues/search/drupal

Screenshot of filtered search interface.

Do a filtered search, follow an issue.

Dreditor

Browser plugin

dreditor.org

Screenshot of Dreditor.

Makes the issue queue easier to work with, go ahead and install it now. Find an issue with a patch, click the review button, click the simply test me button.

Git

Version control

drupal.org/project/drupal/git-instructions

Photo CC-BY-ND Tambako The Jaguar.

Git is the version control system of choice for the Drupal project. It allows us to track all the changes, merge peoples work, easily see changes and history.
